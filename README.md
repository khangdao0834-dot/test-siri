# test-siri

Small static demo project for K-Lab.

## What it is

An educational hash visualizer that shows:

- how input text is split into chunks
- how each chunk updates the rolling state
- the final digest output

## Files

- `index.html`: app shell
- `styles.css`: visual design and layout
- `app.js`: toy hash pipeline and rendering

Open `index.html` in a browser to use it.
